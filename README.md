# 🧠 MindPulse
Plataforma web orientada al bienestar mental y emocional en equipos empresariales.

## 👩‍💻 Integrantes:
- Andrea Murcia – Desarrollo Frontend & Backend, Gestión, contenido y diseño

## 🛠️ Tecnologías
- Frontend: React + Vite
- Backend: Node.js + Express
- Base de datos: MongoDB Atlas

## 📌 Metodología
Scrum – Desarrollo por Sprints

## 🎯 Objetivo
Desarrollar una aplicación web que promueva el bienestar mental en equipos de trabajo mediante servicios digitales.
